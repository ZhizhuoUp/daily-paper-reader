# A Duality-Based Optimization Formulation of Safe Control Design with State Uncertainties 

Xiao Tan ∗, Rahal Nanayakkara ∗, Paulo Tabuada, and Aaron D. Ames 

Abstract — State estimation uncertainty is prevalent in real-world applications, hindering the application of safety-critical control. Existing methods address this by strength-ening a Control Barrier Function (CBF) condition either to handle actuation errors induced by state uncertainty, or to enforce stricter, more conservative sufficient conditions. In this work, we take a more direct approach and formulate a robust safety filter by analyzing the image of the set of all possible states under the CBF dynamics. We first prove that convexifying this image set does not change the set of possible inputs. Then, by leveraging duality, we propose an equivalent and tractable reformulation for cases where this convex hull can be expressed as a polytope or ellipsoid. Simulation results show the approach in this paper to be less conservative than existing alternatives. 

I. I NTRODUCTION 

Safety is one of the most crucial requirements in today’s control applications. Formally, safety is a property that re-quires all trajectories of the system to remain within a pre-scribed safe set. Recently, Control Barrier Functions (CBF) [1] have emerged as a convenient tool to enforce safety guarantees for control systems by constructing an inequality whose sat-isfaction implies the assurance of system safety. Furthermore, for systems which are affine in the control input, the CBF inequality reduces to a linear constraint on the input, allowing fast and easy computation of safe control inputs in real time. Despite its effectiveness, the standard CBF framework as-sumes perfect knowledge of system dynamics and state infor-mation. However, in practical applications, state uncertainty may arise from sensor degradation, environmental interfer-ence, or ill-calibrated state estimation algorithms. Fortunately, advances in observer design provide not only state estimates, but also deterministic bounds on the estimation error [2], [3], [4]. This paper addresses the problem of finding a safe control input given a set of states guaranteed to contain the true state, which can be obtained using these observer bounds. Enforcing safety under these conditions requires satisfying the CBF inequality over the entire set of possible states. This MR-CBF framework is generalized to include uncertainty in the dynamics in [6], and expanded via interval analysis with higher-order Taylor models in [7]. The results in [8], uses a similar technique but also assumes that the control effectiveness gain in the CBF inequality does not change sign, heavily restricting the applicability of this method. An alternative perspective is to consider the effect of state uncertainty on the closed loop control law, providing robust-ness guarantees against the resulting actuation errors [9], [10], [11], [12]. We discuss this in more detail in Section II-B. For stochastic state uncertainties, works such as [13], [14], [15] provide probabilistic safety guarantees, though we do not make direct comparisons as our approach focuses on the deterministic setting. In this work, we take a direct approach to the safe control problem under state uncertainty by designing an optimization-based safety filter that enforces a CBF constraint across the entire set of possible states. This is naturally formulated as a robust optimization problem. We first show that over-approximating this uncertainty set with its convex hull in the proposed robust optimization problem, does not change the set of solutions. We next focus on the case where this convex hull can be expressed as a polytope or ellipsoid, and propose a duality-based reformulation that is both equivalent to the original problem and efficient to solve online. In contrast to prior works such as [5], which rely on Lipschitz constants to provide a simple, albeit coarse, over-approximation of the uncertainty set, our approach accurately computes this required set resulting in a significantly less conservative formulation.

B. A Dynamic Uncertainty Perspective 

In practice, feedback control relies on a state estimate ˆx

rather than the true state x. Given a feedback control law 

k : Rn → Rm, applying the control input u = k(ˆ x) instead of the ideal u = k(x) introduces a dynamic disturbance 

d = k(ˆ x) − k(x), resulting in the closed-loop dynamics: 

˙x = f (x) + g(x)( k(x) + d). (5) When ˆx−x is bounded and the controller k is locally Lipschitz, for any bounded x one concludes that d is bounded. Here d

acts as a matched dynamic uncertainty to the nominal closed-loop system ˙x = f (x) + g(x)k(x). A well-established safety notion in this setting is called Input-to-State Safety (ISSf) [11]. A controller satisfying an ISSf-CBF constraint ensures the forward invariance of an inflated safe set: 

Cd = {x ∈ Rn : h(x) + γ( ¯d) ≥ 0}, (6) under the perturbed dynamics (5), where γ is a class K

function and d = ess sup t|d(t)|. Intuitively, an ISSf-CBF guarantees that the system remains within a neighborhood of the original safe set that scales with the disturbance magnitude. To achieve ISSf style guarantees, the recent work [10] proposes the R-CBF constraint: 

Lf h(x) + Lg h(x)u + α(h(x)) ≥ γ1∥Lg h(x)∥+γ2∥Lg h(x)∥2,

(R-CBF) with tuning parameters γ1, γ 2 ∈ R>0. As established in [10], γ1 and γ2 dictate the disturbance threshold determining whether the original safe set C or an inflated set (6) is rendered forward invariant. Although recent extensions adapt these pa-rameters online [12], it remains unclear how to systematically design γ1 and γ2 to optimize the closed-loop robustness/safety guarantees when the full state uncertainty set is known

In what follows, we propose a direct approach to analyze the exact conditions under which the CBF constraint is satisfied over the entire set of possible states.

V. C ONCLUSION 

In this work, we proposed a duality-based reformulation for robust optimization problems arising from CBF-based safety filter design under state uncertainty. We first demonstrated that over-approximating the uncertainty set by its convex hull does not introduce conservativeness into the robust optimiza-tion problem. For polytopic and ellipsoidal uncertainties, our duality-based reformulations were shown to be equivalent, and their effectiveness was demonstrated using two case studies. This approach effectively eliminates the need for Lipschitz constant estimates in prior results and provides the least conservative solution for a large class of systems. While the requirements for feasibility were discussed, guaranteeing the recursive feasibility of the closed-loop system remains an open problem and will be the subject of future research.
