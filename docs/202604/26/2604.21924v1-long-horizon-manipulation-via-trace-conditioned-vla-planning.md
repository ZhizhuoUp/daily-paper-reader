## 摘要
长程操纵对于视觉-语言-动作（VLA）策略而言仍具挑战性：实际任务通常是多步骤的、依赖进度的，且容易受到复合执行误差的影响。我们提出了 LoHo-Manip，这是一个模块化框架，通过专门的任务管理 VLM 将短程 VLA 执行扩展到长程指令遵循。管理器与执行器解耦，并以退避时界（receding-horizon）方式调用：给定当前观测，它会预测一个感知进度的剩余计划，该计划结合了：(i) 具有明确“已完成+剩余”划分的子任务序列，作为轻量级语言记忆；(ii) 视觉轨迹——一种紧凑的 2D 关键点轨迹提示，用于指定移动路径和接近目标。执行器 VLA 经过调整以根据渲染的轨迹进行条件化，从而通过遵循轨迹将长程决策转化为重复的局部控制。至关重要的是，在每一步预测剩余计划会产生一个隐式闭环：失败的步骤会保留在后续输出中，轨迹也会相应更新，从而实现自动延续和重新规划，而无需手工设计的恢复逻辑或脆弱的视觉历史缓存。在仿真和真实 Franka 机器人上进行的广泛实验（涵盖具身规划、长程推理、轨迹预测和端到端操纵）表明，该方法在长程成功率、鲁棒性和分布外泛化方面取得了显著提升。项目页面：https://www.liuisabella.com/LoHoManip

## Abstract
Long-horizon manipulation remains challenging for vision-language-action (VLA) policies: real tasks are multi-step, progress-dependent, and brittle to compounding execution errors. We present LoHo-Manip, a modular framework that scales short-horizon VLA execution to long-horizon instruction following via a dedicated task-management VLM. The manager is decoupled from the executor and is invoked in a receding-horizon manner: given the current observation, it predicts a progress-aware remaining plan that combines (i) a subtask sequence with an explicit done + remaining split as lightweight language memory, and (ii) a visual trace -- a compact 2D keypoint trajectory prompt specifying where to go and what to approach next. The executor VLA is adapted to condition on the rendered trace, thereby turning long-horizon decision-making into repeated local control by following the trace. Crucially, predicting the remaining plan at each step yields an implicit closed loop: failed steps persist in subsequent outputs, and traces update accordingly, enabling automatic continuation and replanning without hand-crafted recovery logic or brittle visual-history buffers. Extensive experiments spanning embodied planning, long-horizon reasoning, trajectory prediction, and end-to-end manipulation in simulation and on a real Franka robot demonstrate strong gains in long-horizon success, robustness, and out-of-distribution generalization. ### 1. 核心问题与研究动机
*   **核心问题**：现有的视觉-语言-动作（VLA）模型在处理长程任务时面临三大挑战：
    1.  **复合误差**：多步任务中微小的执行偏差会随时间累积，导致最终失败。
    2.  **进度感知缺失**：模型难以准确判断当前任务进行到了哪一步，以及下一步该做什么。
    3.  **泛化性差**：端到端模型往往对训练分布高度敏感，难以处理未见过的物体或复杂的逻辑指令。
*   **研究背景**：传统的单体式（Monolithic）模型试图将规划与控制耦合在一起，但在面对需要数十步操作的家庭任务（如“清理桌面并分类”）时表现脆弱。

### 2. 方法论（LoHo-Manip 框架）
核心思想是将**高层任务管理**与**底层动作执行**解耦，通过“视觉轨迹”作为两者之间的桥梁。
*   **任务管理器（VLM）**：
    *   基于预训练的视觉语言模型（如 Qwen3-VL），负责高层推理。
    *   **输入**：当前观测图像、用户指令、以及轻量级的文本历史记忆（已完成的子任务）。
    *   **输出**：预测“剩余计划”，包含：
        1.  **子任务序列**：明确划分“已完成”和“待完成”的步骤。
        2.  **视觉轨迹（Visual Trace）**：在 2D 图像上生成一系列关键点，指明机器人末端执行器应该移动的路径和目标物体。
*   **执行器（VLA）**：
    *   基于 π0.5 等 VLA 模型，负责短程控制。
    *   **核心技术**：执行器被微调为“轨迹条件化”，即它会接收管理器生成的视觉轨迹作为提示（Prompt），通过“跟随轨迹”来完成具体的物理交互。
*   **闭环机制（Receding-Horizon）**：
    *   系统以退避时界方式运行，定期调用管理器更新计划。如果某一步失败（如抓取落空），管理器会在下一帧观测中识别到状态未变，从而自动重新生成相同的子任务和轨迹，实现隐式的错误恢复。

### 6. 主要结论与发现
*   **解耦优于耦合**：将规划与执行分开，能显著提升长程任务的鲁棒性，尤其是在处理复合误差时。
*   **视觉轨迹的有效性**：2D 轨迹作为一种“空间提示”，比纯文本指令更能有效地引导底层 VLA 模型进行精确操作。
*   **隐式恢复能力**：无需手动编写复杂的纠错逻辑，仅靠闭环的“剩余计划预测”即可实现自动重试和路径修正。
*   **强泛化性**：即使底层执行器未见过某些物体，只要高层管理器能正确“指点”出物体位置，执行器就能完成任务。

### 7. 优点（亮点）
*   **模块化设计**：管理器与执行器相互独立，可以轻松更换更强大的 VLM 或不同构型的机器人执行器。
*   **轻量级记忆**：使用文本形式的“已完成/剩余”列表代替沉重的视频历史缓存，既节省算力又减少了分布偏移。
*   **数据驱动的纠错**：通过合成失败样本进行训练，使模型具备了识别语义错误（如抓错物体）并主动纠正的能力。

### 8. 不足与局限
*   **维度限制**：目前的视觉轨迹主要基于 **2D 像素坐标**，对于需要高度 3D 空间感知或复杂接触力控制的任务（如插拔、精密组装）可能信息量不足。
*   **感知依赖**：系统的成功高度依赖于管理器对物体的视觉定位精度，如果 VLM 在复杂遮挡下定位失败，整个链条会断裂。
*   **场景局限**：实验主要集中在桌面操纵（Tabletop Manipulation），尚未在更广阔的移动操纵或动态多机协作场景中验证。
